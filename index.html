Quickstart

Chạy được trong 5 phút

Không cần account doanh nghiệp. Bắt đầu với tier miễn phí, nâng cấp khi cần.

TypeScript 
import { AIPClient } from '@aip/sdk'

// Khởi tạo client
const aip = new AIPClient({
  apiKey: process.env.AIP_API_KEY,
  region: 'ap-southeast-1',
})

// Gọi AI model
const response = await aip.chat.complete({
  model: 'claude-sonnet-4-6',
  messages: [{ role: 'user', content: 'Xin chào!' }],
  maxTokens: 1024,
})

console.log(response.content) // → 'Xin chào! Tôi có thể giúp gì cho bạn?'
